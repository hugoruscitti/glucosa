Glocosa, azúcar para crear juegos

Glocosa es una biblioteca para hacer videojuegos. Su diseño es tan básico, que cabe completamente en un archivo .py.

El objetivo de Glucosa, es funcionar sobre cairo y gtk para construir juegos muy sencillos para equipos XO (de olpc) y facilitar la transición de pygame a cairo.

¿Por qué existe glucosa?

Ok, es cierto, hay un montón de bibliotecas para hacer juegos. Pero hay un motivo real para crear otra mas... Actualmente pygame no funciona correctamente sobre los equipos XO, es una pena, pero ciertamente no integra bien con GTK y el ecosistema de sugar.

La forma favorita de hacer aplicaciones para sugar es usar pygtk junto a cairo, para hacer dibujos multimedia.

Pero... aquí el dilema, pygame no funciona bien sobre sugar y cairo es un poco difícil de manipular cuando quieres hacer un juego.

Aquí nace la idea de crear glucosa, ¡Una super extra mini-biblioteca orientada a crear juegos sobre cairo y gtk !

Instalación

En un sistema Ubuntu, necesitas tener instalado pygtk y pycairo para comenzar. Luego, tienes que descargar glucosa y un pequeño ejemplo: 

wget https://raw.github.com/hugoruscitti/glucosa/master/glucosa.py
wget https://raw.github.com/hugoruscitti/glucosa/master/demo.py
Listo, ya está instalado. Ahora puedes probar nuestro ejemplo: 
python demo.py
Otra opción de instalación, es descargarte el repositorio completo y explorar el resto de los archivos: 
git clone git@github.com:hugoruscitti/glucosa.git
